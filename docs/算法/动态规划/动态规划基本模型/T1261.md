# 🏆 题解：城市交通路网（动态规划求最短路径）

## 📌 1. 题目剖析
给定一个城市交通网的费用矩阵（单向通行），我们需要求解从起点（城市 1）到终点（城市 $N$）的**最少总费用**，并且打印出具体的**行车路线**。

* **核心痛点**：贪心算法在这里会失效。因为眼前的路即使很便宜，也可能把你引向后面的一家“黑店”（例如样例中 1->4 只要花费 1，但后续 4->6 却高达 12）。
* **破局思路**：使用动态规划（DP），记录到达每一个中转城市的最少历史总花费，稳扎稳打推到终点。

## 🧠 2. 核心算法设计

我们需要两个数组来打辅助：
1.  **`dp[i]`**：表示从起点（城市 1）到达城市 `i` 的最少总费用。
    * *初始化*：`dp[1] = 0`（站在起点不花钱），其余 `dp[i] = ∞`（设为极大值，方便后续求最小值）。
2.  **`pre[i]`**：表示到达城市 `i` 的“最优前一站”是哪个城市（用于回溯路线）。
    * *初始化*：全为 0。

**🔄 状态转移方程（货比三家）：**
当我们想求到达城市 `i` 的最少费用时，我们要把所有能直接到达 `i` 的前置城市 `j` 都看一遍：
`dp[i] = min(dp[i], dp[j] + a[j][i])`
*(前提：城市 `j` 到城市 `i` 之间有路，即 `a[j][i] > 0`)*

## 📊 3. 高能预警：DP 状态表全景推导

这是本题解的最核心部分。我们以样例数据为例，一步步模拟系统是如何避开“陷阱”，找出最优解的。

*初始状态：`dp[1] = 0`，其余为 `∞`*

| 目标城市 `i` | 考察所有的“前一站” `j` (计算 `dp[j] + a[j][i]`) | 最终选定的最优 `j` | 更新后的 `dp[i]` (最少花费) | 记录前任 `pre[i]` | 💡 讲师解析 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **2** | `j=1`: 0 + 2 = 2 | 1 | **2** | **1** | 只有起点能到 2 |
| **3** | `j=1`: 0 + 5 = 5 | 1 | **5** | **1** | 只有起点能到 3 |
| **4** | `j=1`: 0 + 1 = 1 | 1 | **1** | **1** | 只有起点能到 4 |
| **5** | `j=2`: 2 + 12 = 14<br>`j=3`: 5 + 6 = 11<br>`j=4`: 1 + 13 = 14 | **3** | **11** | **3** | 重点！从 3 走虽然起点贵，但总价 11 完胜其他路线！ |
| **6** | `j=2`: 2 + 14 = 16<br>`j=3`: 5 + 10 = 15<br>`j=4`: 1 + 12 = 13 | **4** | **13** | **4** | 去 6 号城市，从 4 走最便宜。 |
| **7** | `j=3`: 5 + 4 = 9<br>`j=4`: 1 + 11 = 12 | **3** | **9** | **3** | 从 3 走最划算。 |
| **8** | `j=5`: 11 + 3 = 14<br>`j=6`: 13 + 6 = 19 | **5** | **14** | **5** | 去 8 只能经由 5 或 6。 |
| **9** | `j=5`: 11 + 9 = 20<br>`j=6`: 13 + 5 = 18<br>`j=7`: 9 + 10 = 19 | **6** | **18** | **6** | 去 9 的路线多，对比发现走 6 最好。 |
| **10(终点)** | `j=8`: 14 + 5 = 19<br>`j=9`: 18 + 2 = 20 | **8** | **19** | **8** | **大结局！** 走 8 号城市只需 19 块！ |

### ⏪ 4. 终极倒带：寻找隐藏的路线
现在我们知道了最省费用是 **19**，那路线是怎么来的呢？利用 `pre` 数组像顺藤摸瓜一样倒推：

1. 从终点 **10** 开始看：`pre[10] = 8` (前一站是 8)
2. 转移到 **8** 看：`pre[8] = 5` (前一站是 5)
3. 转移到 **5** 看：`pre[5] = 3` (前一站是 3)
4. 转移到 **3** 看：`pre[3] = 1` (前一站是起点 1)
5. 转移到 **1**：到达起点，倒推结束！

**倒序路径**：`10 -> 8 -> 5 -> 3 -> 1`
**反转得到最终答案**：`1 3 5 8 10`

## 💻 5. 标准 C++ 代码实现

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int n;
    if (!(cin >> n)) return 0;

    // a 数组存储图的邻接矩阵，下标 1~n
    vector<vector<int>> a(n + 1, vector<int>(n + 1, 0));
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            cin >> a[i][j];
        }
    }

    // dp[i] 初始化为极大值，代表一开始都不可达
    vector<int> dp(n + 1, 999999); 
    vector<int> pre(n + 1, 0);
    dp[1] = 0; // 起点费用为 0

    // 核心 DP 过程
    for (int i = 2; i <= n; i++) {
        for (int j = 1; j < i; j++) {
            if (a[j][i] > 0) { // 如果 j 到 i 有通路
                if (dp[j] + a[j][i] < dp[i]) {
                    dp[i] = dp[j] + a[j][i]; // 更新最小花费
                    pre[i] = j;              // 记录最优前驱节点
                }
            }
        }
    }

    cout << "minlong=" << dp[n] << endl;

    // 利用 pre 数组回溯路径
    vector<int> path;
    int curr = n;
    while (curr != 0) {
        path.push_back(curr);
        curr = pre[curr];
    }

    // 倒序输出真实路线
    for (int i = path.size() - 1; i >= 0; i--) {
        cout << path[i] << (i == 0 ? "" : " ");
    }
    cout << endl;

    return 0;
}
```
