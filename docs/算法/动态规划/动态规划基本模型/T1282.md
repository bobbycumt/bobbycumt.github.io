### T1282最大子矩阵和

[▶ 查看动画](/html/T1282.html)

**Kadane 算法（卡丹算法）** 是计算机科学中一个极其经典且优雅的算法，它专门用来解决 **“最大连续子数组和”** 问题。

在刚才的二维矩阵降维过程中，我们把几行“压扁”成了一维数组，然后对这个一维数组求最大子段和，用的就是 Kadane 算法。

实际上，Kadane 算法就是 **动态规划（DP）** 在最简单的一维空间里的一次完美展现。由于它只需要扫一遍数组（时间复杂度 $O(N)$），并且不需要额外的数组空间（空间复杂度 $O(1)$），它被认为是该问题的最优解。

为了让你秒懂它的精髓，我们先抛开代码，用一个极其通俗的“现实生活”例子来理解。

---

### 🎒 核心哲学：“负资产直接丢弃”

假设你在一条路上闯关捡金币，路上有些格子里是正金币（+），有些是倒扣钱的炸弹（-）。规则是：**你必须连续走一段路，不能跳跃，求在哪一段路能赚到最多的钱。**

你在走的时候，手里会拿着一个“存钱罐”（记录当前连续累加的钱 `current_sum`）：

1.  **如果存钱罐里是正数（赚钱的）**：哪怕你前面遇到一个小炸弹（比如 -1），你也会咬牙走过去。因为你手里的底子厚（比如手里有 5，遇到 -1 变成 4），加上前面的财富，你依然是正资产，未来如果遇到大金币，还能继续滚雪球。
2.  **如果存钱罐里变成负数了（破产了）**：这时候如果你继续带着这个“负资产”往下走，它只会拖累你！无论你下一个遇到的是金币还是炸弹，带着前面的负债，总收益一定会变少。**所以，最明智的决策是：立刻扔掉过去的存钱罐（清零），从脚下这一步重新开始计算！**

这就是 Kadane 算法的全部核心思想：**过去的总和如果是累赘（小于 0），就果断割肉抛弃，从当前重新开始。**

---

### 📐 严谨的 DP 状态转移方程

我们将上面的哲学翻译成数学语言（动态规划）：

* **状态定义**：设 $dp[i]$ 表示—— **“以第 $i$ 个元素为结尾”** 的所有连续子数组中，最大的那个和。
* **状态转移方程**：
    当我们站在第 $i$ 个元素 $A[i]$ 上时，我们面临两种选择：
    1.  **和前面的连起来**：总和是 $dp[i-1] + A[i]$。
    2.  **自己单干（割裂前面的）**：总和就是 $A[i]$ 自己。

    所以方程是：
    $$dp[i] = \max(A[i], dp[i-1] + A[i])$$
    *（仔细看这个公式，如果 $dp[i-1]$ 是正的，那加上它肯定更大；如果 $dp[i-1]$ 是负的，那 $A[i]$ 自己单干肯定比加上负数要大，完美契合了刚才的“负资产丢弃”理论！）*

* **空间压缩（为什么常被称为 $O(1)$ DP）**：
    你会发现，$dp[i]$ 的计算**只依赖于它前面的那一个状态** $dp[i-1]$。我们根本不需要开一个长度为 $N$ 的数组去存过去的历史！
    我们只需要一个变量 `current_sum` 来扮演 $dp[i-1]$ 的角色，再用一个变量 `max_sum` 来记录历史最高纪录（全局最大值）就可以了。

### 💻 极简代码模板

在算法竞赛和面试中，Kadane 算法是非常高频的考察点。由于省去了 DP 数组，它的代码看起来极其简洁干练（通常只有不到 10 行核心代码）：


```cpp
#include <iostream>
#include <vector>
#include <algorithm> // 为了使用 max 函数

using namespace std;
int n;
vector<int> nums;
int main() {
    
    cin>>n;
    for(int i=0;i<n;i++){
        int t;
        cin>>t;
        nums.push_back(t);
    }
    int current_sum = nums[0]; // 对应 dp[i-1]
    int max_sum = nums[0];     // 记录全局最大值

    // 从第二个元素开始遍历
    for (int i = 1; i < nums.size(); ++i) {
        // 核心 DP 状态转移：是和前面连起来，还是自己单干？
        // 这里也可以写成：current_sum = (current_sum > 0) ? current_sum + nums[i] : nums[i];
        current_sum = max(nums[i], current_sum + nums[i]);
        
        // 随时挑战全局历史最高纪录
        max_sum = max(max_sum, current_sum);
    }

    cout<<max_sum;
    return 0;
}

```
