## 九九乘法表题解

[▶ 查看动画](/html/九九乘法表.html)

### 一、题目要求
输出下三角形式的九九乘法表：第 `i` 行输出 `i*1` 到 `i*i`。

---

### 二、考查的语法和技能

这道题非常经典，核心考点有：

1. **双重（嵌套）循环**
   - 外层循环控制“行”（`i = 1..9`）
   - 内层循环控制“列”（`j = 1..i`）

2. **格式化输出**
   - 乘积的对齐通常用宽度控制（如 `printf("%2d")`）
   - 同一行中多个算式之间要有空格

3. **换行时机**
   - 每一行打印完成后再换行

4. **边界控制**
   - 内层循环上界是 `i`，不是固定的 9

---

### 三、思路解析

#### 1）先看行
第 1 行只有 1 个算式，第 2 行有 2 个算式……第 9 行有 9 个算式。

所以外层循环 `i` 表示“当前第几行”，范围 `1~9`。

#### 2）再看列
在第 `i` 行里，需要打印：

- `i*1`
- `i*2`
- ...
- `i*i`

所以内层循环 `j` 的范围是 `1~i`。

#### 3）格式细节
每个算式格式是：

```text
i*j=乘积
```

为了和示例更接近，乘积部分用 `%2d`，让个位数前面补空格，两位数保持原样。

#### 4）何时换行
内层循环结束后说明这一行打印完毕，执行一次换行。

---

### 四、带有详细注释的代码（C++）

```cpp
#include <cstdio>

int main() {
    // 外层循环控制行号 i：从第 1 行到第 9 行
    for (int i = 1; i <= 9; i++) {

        // 内层循环控制当前行中的第几个算式 j：1 到 i
        for (int j = 1; j <= i; j++) {

            // 打印一个算式：i*j=乘积
            // %2d 表示宽度为 2，个位数会在前面补一个空格，便于对齐
            std::printf("%d*%d=%2d", i, j, i * j);

            // 不是本行最后一个算式时，补一个空格作为分隔
            if (j != i) {
                std::printf(" ");
            }
        }

        // 一整行输出结束后再换行
        std::printf("\n");
    }

    return 0;
}
```

---

### 五、易错点解析

1. **内层循环写成 `j <= 9`**
   - 这样会打印成“完整 9 列矩形”，而不是下三角。
   - 正确应是 `j <= i`。

2. **把换行写在内层循环里**
   - 会导致每个算式都单独占一行。
   - 正确位置：内层循环结束后。

3. **漏掉算式之间的空格**
   - 输出会粘在一起，格式不符合题意。

4. **格式控制不统一**
   - 若不使用 `%2d`，虽然逻辑正确，但容易和示例格式不一致。

---

### 六、复杂度分析

- 总共打印 `1 + 2 + ... + 9 = 45` 个算式。
- 时间复杂度：`O(9^2)`（常量规模，可视为 `O(1)`）。
- 空间复杂度：`O(1)`。
