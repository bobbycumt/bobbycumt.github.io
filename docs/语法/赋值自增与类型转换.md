## 赋值自增与类型转换

[▶ 查看动画](/html/赋值自增与类型转换.html)

### 一、考查的语法和技能

1. 赋值运算符 `=` 的执行方向：先算右边，再覆盖左边变量。  
2. 复合运算符 `+= -= *= /= %=` 的等价写法与整体性。  
3. 自增自减 `++ --` 的前置/后置差异。  
4. 隐式类型转换（向高精度对齐）与强制类型转换。  
5. 整数除法与浮点除法区别，避免精度陷阱。

---

### 二、思路解析

#### 1）先统一“赋值”认知

- `x = x + 1;` 不是“数学相等”，而是“把新值写回 x”。
- 程序执行时遵循：
  1. 计算右侧表达式；
  2. 把结果写入左侧变量。

#### 2）再掌握“复合运算”的缩写逻辑

- `a += b` 等价于 `a = a + b`。  
- 右侧表达式是一个整体：`a *= b + 3` 等价于 `a = a * (b + 3)`。

#### 3）重点区分前置/后置自增

- `++a`：先变后用。  
- `a++`：先用后变。  
- 建议避免在复杂表达式里混用多个 `++`，可读性和可预测性更高。

#### 4）最后处理类型转换

- 隐式转换：`char -> int -> float -> double`。  
- 强制转换：`(int)3.98` 结果是 `3`（直接截断，不是四舍五入）。  
- 经典陷阱：`5 / 2 == 2`，`5 / 2.0 == 2.5`。

### 三、带详细注释的代码

```cpp
#include <iostream>
using namespace std;

int main() {
    // 1) 赋值与复合运算
    int a = 10;
    a = a + 2;   // 先算右边：10+2=12，再覆盖写回 a
    a *= 3 + 1;  // 等价于 a = a * (3+1)，即 12*4=48

    // 2) 前置与后置自增
    int x = 5;
    int y = ++x; // 先让 x 变 6，再把 6 赋给 y

    int p = 5;
    int q = p++; // 先把 5 赋给 q，再让 p 变 6

    // 3) 隐式类型转换
    int n = 5;
    double d = 2.0;
    double r1 = n / d; // n 自动提升为 double，结果 2.5

    // 4) 强制类型转换
    double pi = 3.98;
    int cut = (int)pi; // 小数直接截断，cut=3

    // 5) 整数除法陷阱
    double r2 = 5 / 2;   // 先做整数除法，结果 2，再转 double -> 2.0
    double r3 = 5 / 2.0; // 浮点参与运算，结果 2.5

    cout << "a=" << a << "\n";
    cout << "x=" << x << ", y=" << y << "\n";
    cout << "p=" << p << ", q=" << q << "\n";
    cout << "r1=" << r1 << ", cut=" << cut << "\n";
    cout << "r2=" << r2 << ", r3=" << r3 << "\n";

    // 6) 综合练习
    int u = 10;
    int v = u++; // v=10, u=11
    v += ++u;    // ++u 后 u=12, v=10+12=22

    cout << "u=" << u << ", v=" << v << "\n";
    return 0;
}
```

---

### 四、易错点解析

1. 把 `=` 当成“相等”，忽略“覆盖写回”的本质。  
2. 误读 `a *= b + 3`，漏掉括号语义。  
3. 混淆 `++a` 与 `a++` 的生效时机。  
4. 忘记整数除法会截断：`5/2` 不是 `2.5`。  
5. 把强制转换当成四舍五入：`(int)3.98` 实际是 `3`。

### 五、考试常见技巧

- **四舍五入到整十**：`(X + 5) / 10 * 10`。  
- **数位分离**：`N % 10` 取个位，`N / 10` 去个位。可结合 [[数位分离#固定位数拆位]] 复习。
