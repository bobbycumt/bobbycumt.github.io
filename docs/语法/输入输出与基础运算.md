## 输入输出与基础运算（cin/cout、int、+-*/%）

[▶ 查看动画](/html/输入输出与基础运算.html)

### 一、考查的语法和技能

1. **`int` 类型**
   - 声明整型变量：`int a, b;`
   - 理解 `int` 适合整数运算（不含小数部分）。

2. **输入 `cin`**
   - 基本写法：`cin >> a;`
   - 连续输入：`cin >> a >> b;`

3. **输出 `cout`**
   - 基本写法：`cout << a;`
   - 连续输出：`cout << a << " " << b << "\n";`

4. **数学运算符 `+ - * / %`**
   - 加、减、乘、整除、取模。
   - 重点：`/` 在 `int` 间是**整除**，`%` 仅用于整数。

5. **使用 `cin/cout` 连续输入输出**
   - 一行读取多个数据。
   - 一行输出多个结果（注意空格与换行）。

---

### 二、思路解析

这类基础题通常流程固定：

1. 定义变量：`int a, b;`
2. 输入数据：`cin >> a >> b;`
3. 计算五种结果：
   - `sum = a + b`
   - `diff = a - b`
   - `prod = a * b`
   - `quot = a / b`（整除）
   - `rem = a % b`
4. 使用 `cout` 一次性规范输出。

---

### 三、带有详细注释的代码（C++）

```cpp
#include <iostream>
using namespace std;

int main() {
    // 1) 定义整型变量：a、b 表示输入的两个整数
    int a, b;

    // 2) 连续输入：从标准输入依次读取 a 和 b
    cin >> a >> b;

    // 3) 基本运算
    int sum  = a + b; // 加法
    int diff = a - b; // 减法
    int prod = a * b; // 乘法

    // 注意：这里假设 b != 0
    // 对 int 而言，/ 是整除（小数部分会被舍去）
    int quot = a / b; // 整除

    // % 是取模（余数），只适用于整数
    int rem  = a % b;

    // 4) 连续输出，注意格式与换行
    cout << "a+b=" << sum  << "\n";
    cout << "a-b=" << diff << "\n";
    cout << "a*b=" << prod << "\n";
    cout << "a/b=" << quot << "\n";
    cout << "a%b=" << rem  << "\n";

    return 0;
}
```

### 四、易错点解析

1. **把 `=` 和 `==` 混淆**
   - `=` 是赋值，`==` 才是比较。

2. **忽略整除特性**
   - `7 / 2` 在 `int` 中结果是 `3`，不是 `3.5`。

3. **`%` 用在非整数上**
   - `double` 不能直接用 `%`。

4. **除数为 0**
   - `a / 0`、`a % 0` 都是非法，必须提前判断。

5. **输出格式错误**
   - 少空格、少换行、顺序错，都会导致判题错误。

### 五、课堂练习建议

- 输入两个整数，输出它们的和、差、积、商、余数。
- 输入三个整数 `a b c`，输出 `(a+b)*c` 与 `a%(b+c)`。
- 练习连续输入输出：一行输入，分多行输出；或多行输入，一行输出。
