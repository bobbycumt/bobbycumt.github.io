2. 赋值与复合运算

写法等价形式说明
a += ba = a + b加后写回
a *= b + 3a = a * (b + 3)右侧是整体

口诀：先算右边，再写左边。

5. 综合练习答案

int x = 10; int y = x++; y += ++x;

  • 第一步：y = x++ 后，y=10, x=11。
  • 第二步：y += ++x，先 x=12，再 y=10+12=22。

最终：x=12，y=22

避坑：不要在复杂表达式中堆叠多个自增。
